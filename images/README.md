# Guía de Imágenes — Dra. Sandra Molano

Si usas nombres diferentes, actualiza las rutas en `index.html`.

## Notas
- Formatos aceptados: `.jpg`, `.jpeg`, `.png`, `.webp`
- Para mejor rendimiento, comprime las imágenes antes de subirlas (puedes usar [squoosh.app](https://squoosh.app))
- Mientras no tengas las imágenes, la página muestra placeholders dorados automáticamente.
